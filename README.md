# Лабораторная работа 5.1: Notebook

Одностраничное PHP-приложение для управления списком контактов с хранением в MySQL.

### Требования

- **`index.php`** – точка входа, подключает модули в зависимости от параметров URL.
- **`menu.php`** – меню с кнопками:
  - Просмотр
  - Добавление записи
  - Редактирование записи
  - Удаление записи  
  Активный пункт выделяется красным, остальные – синим. В режиме «Просмотр» появляется подменю сортировки (по ID, фамилии, дате рождения) кнопками меньшего размера, выделение основного пункта сохраняется.
- **`viewer.php`** – таблица всех контактов с пагинацией (≤10 записей на страницу). Ссылки пагинации при наведении получают рамку 2px.
- **`add.php`** – форма добавления контакта (фамилия, имя, отчество, пол, дата рождения, телефон, адрес, email, комментарий). После отправки выводится зелёное сообщение об успехе или красное об ошибке.
- **`edit.php`** – список контактов (ссылки с фамилией и именем, сортировка по фамилии/имени). При выборе форма заполняется данными, текущий контакт выделен.
- **`delete.php`** – список ссылок (фамилия и инициалы). При клике запись удаляется, выводится сообщение с фамилией удалённого.
- **База данных** – таблица `contacts` со столбцами:  
  `id`, `surname`, `name`, `lastname`, `gender`, `birthdate`, `phone`, `address`, `email`, `comment`, `created_at`.
