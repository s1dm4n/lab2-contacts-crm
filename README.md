# Лабораторная работа №2: Contacts CRM

## Описание

Веб-приложение "Записная книжка" для управления контактами. Реализована архитектура с модульной структурой, позволяющая управлять всеми аспектами работы с базой данных контактов.

## 📋 Требования задания

### Структура файлов
- `index.php` — единственный загружаемый документ (точка входа)
- `modules/menu.php` — формирование меню и его работа
- `modules/viewer.php` — вывод данных с пагинацией и сортировкой
- `modules/add.php` — добавление новой записи
- `modules/edit.php` — редактирование записи
- `modules/delete.php` — удаление записи

### Функциональность

✅ **Просмотр контактов** с таблицей (max 10 строк на странице)  
✅ **Пагинация** — ссылки на страницы с номерами  
✅ **Сортировка** — по дате добавления, фамилии, дате рождения  
✅ **Добавление** — форма с валидацией  
✅ **Редактирование** — выбор из списка + редактирование  
✅ **Удаление** — список контактов с подтверждением  
✅ **Активное меню** — выделение красным цветом активного пункта  

### Атрибуты контакта
- Фамилия ✓
- Имя ✓
- Отчество
- Пол (мужской/женский)
- Дата рождения
- Телефон
- Адрес
- Email
- Комментарий

## 🚀 Запуск

### Локально (XAMPP)
1. Убедитесь что Apache и MySQL запущены в XAMPP
2. Откройте: `http://localhost/lab2-contacts-crm/`

### На хостинге
1. Загрузите файлы по FTP
2. Откройте в браузере

## 🗄️ База данных

При первом запуске автоматически создаются:
- БД: `contacts_db`
- Таблица: `contacts` с полями контакта

```sql
CREATE TABLE `contacts` (
    `id` INT AUTO_INCREMENT PRIMARY KEY,
    `surname` VARCHAR(100) NOT NULL,
    `name` VARCHAR(100) NOT NULL,
    `lastname` VARCHAR(100),
    `gender` ENUM('мужской','женский'),
    `birthdate` DATE,
    `phone` VARCHAR(20),
    `address` TEXT,
    `email` VARCHAR(150),
    `comment` TEXT,
    `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

- **MySQL** — база данных
